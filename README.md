# 🐱 喵言喵语插件

## 🌸 插件简介
喵~喵~喵~欢迎使用喵言喵语插件！这是一个专为AstrBot开发的卡拉彼丘猫娘经典语录插件喵~ ✨

该插件可以在QQ聊天中随机发送卡丘猫娘的经典语录，让你的聊天更加有趣喵~ 🎉

## 🌟 功能特性

- **随机语录**：每次发送命令都会随机显示一条卡丘猫娘经典语录喵~
- **卡丘表情包**：发送"/卡丘表情包"命令可以随机获取一张可爱的卡丘表情包喵~
- **自动更新**：每小时自动检查GitHub仓库的喵言喵语数据更新，确保使用最新内容喵~
- **数据丰富**：从meowQuotes.js文件读取大量经典喵语数据喵~
- **命令简单**：只需发送"喵言喵语"或"/卡丘表情包"命令即可使用对应功能喵~

## 🚀 使用说明

1. **启用插件**：在AstrBot中启用喵言喵语插件喵~
2. **发送语录命令**：在QQ聊天窗口中发送 `喵言喵语` 命令喵~
3. **接收语录**：机器人会自动从预设的喵言喵语内容库中随机选择一条内容，并发送到当前聊天窗口喵~
4. **发送表情包命令**：在QQ聊天窗口中发送 `/卡丘表情包` 命令喵~
5. **接收表情包**：机器人会自动从emoji目录中随机选择一张卡丘表情包，并发送到当前聊天窗口喵~
