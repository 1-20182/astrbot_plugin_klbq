# AstrBot Plugin Publishing - Verification Checklist

- [ ] Metadata file contains all required fields (name, display_name, desc, version, author, repo)
- [ ] Plugin name follows the astrbot_plugin_ prefix convention
- [ ] Version follows semantic versioning format
- [ ] Main plugin file (main.py) exists and is correctly structured
- [ ] Metadata file (metadata.yaml) exists and is correctly formatted
- [ ] Data directory exists and contains required files
- [ ] Emoji directory exists and contains emoji files
- [ ] Plugin follows AstrBot's directory structure requirements
- [ ] Documentation is complete and up-to-date
- [ ] Plugin has been tested and functions correctly
- [ ] AstrBot plugin publishing process has been identified
- [ ] Plugin has been submitted to the correct AstrBot repository
- [ ] Plugin installs successfully in AstrBot
- [ ] "喵言喵语" command works correctly
- [ ] "卡丘表情包" command works correctly
- [ ] Auto-update functionality works as expected
